Configure your ESC/POS thermal receipt printer. Most bar printers connect over the network (WiFi/Ethernet) or via USB. Leave everything blank to use only the browser print fallback.

Find your IDs on Windows: Device Manager → Universal Serial Bus devices. On Linux: lsusb.

How to set up a receipt printer

  1. Run pip install python-escpos and restart the app.
  2. For network printers: connect the printer to your WiFi, find its IP in its menu, enter it above.
  3. For USB on Windows: plug in the printer, open Device Manager, find the USB device, note the Vendor ID and Product ID from the hardware details tab.
  4. For USB on Linux/Mac: run lsusb in terminal. You'll see lines like ID 04b8:0202 Seiko Epson Corp.
  5. Save settings, then go to any order page and click 🖨 Print Receipt to test.
